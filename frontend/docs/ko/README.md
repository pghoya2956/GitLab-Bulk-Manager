# GitLab Bulk Manager

GitLab 그룹과 프로젝트를 효율적으로 대량 관리할 수 있는 웹 애플리케이션입니다.

## 주요 기능

### 리소스 관리
- **생성**: YAML 기반으로 그룹과 프로젝트를 대량 생성
- **삭제**: 선택한 항목을 일괄 삭제
- **이동**: 프로젝트나 그룹을 다른 그룹으로 이동
- **복제**: 구조와 설정을 포함하여 복제
- **아카이브**: 프로젝트 아카이브 및 해제

### 설정 및 구성
- **설정 변경**: 가시성, 설명 등 일괄 변경
- **멤버 관리**: 멤버 추가/제거 및 권한 변경
- **보호 규칙**: 브랜치 보호 및 MR 승인 규칙 설정
- **CI/CD 설정**: 파이프라인 설정 동기화

### 고급 기능
- **이슈/MR**: 이슈 및 Merge Request 관리

## 시작하기

1. GitLab Personal Access Token을 준비합니다
2. 로그인 페이지에서 GitLab URL과 토큰을 입력합니다
3. 왼쪽 트리에서 관리할 그룹/프로젝트를 선택합니다
4. 원하는 작업 카드를 클릭하여 대량 작업을 수행합니다

## 팁

- **다중 선택**: 체크박스를 사용하거나 행을 클릭하여 여러 항목을 선택할 수 있습니다
- **드래그 앤 드롭**: 선택한 항목을 다른 그룹으로 드래그하여 이동할 수 있습니다
- **아카이브 필터**: 프로젝트 필터를 사용하여 활성/아카이브 프로젝트를 필터링할 수 있습니다

## 필요한 권한

대부분의 작업에는 해당 그룹/프로젝트에 대한 **Maintainer** 이상의 권한이 필요합니다.

| 작업 | 필요 권한 |
|------|----------|
| 조회 | Reporter 이상 |
| 설정 변경 | Maintainer 이상 |
| 삭제 | Owner |
| 멤버 관리 | Maintainer 이상 |
